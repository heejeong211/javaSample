0. File의 경로(pathname)과 파일 이름(Filename)
   - pathname은 절대 경로와 상대경로가 존재
   - filename은 보통 파일 자체의 이름을 말함

1. File 사용하기
   - PrintWriter 실습
     import java.io.*
     
     try {
         PrintWriter printer = new PrinterWriter("Hello.txt");

         printer.println("Hello World");
 
         printer.flush();
         printer.close();
    } catch (Exception ex) {
    }
